# Програмна система автоматизації діяльності закладу харчування "Гурман"

Даний проєкт є курсовою роботою, розробленою на базі технології **ASP.NET Core MVC**. Система призначена для автоматизації процесів обслуговування клієнтів, керування меню та фінансового обліку в закладах ресторанного господарства.

## Ключові можливості
- **Автоматизація замовлень:** Створення замовлень, прив'язка до столиків.
- **Керування меню:** Адміністрування переліку страв, категорій (FoodTypes) та контроль наявності.
- **Аналітичний модуль:** Автоматичний розрахунок прибутку за обраний період (фільтрація за датою).
- **Рольова модель:** Різні рівні доступу для Менеджерів, Офіціантів та Клієнтів.

## Тестові дані для входу (Credentials)

Для перевірки функціоналу системи в базу даних автоматично додані наступні користувачі:

| Роль      | Логін      | Пароль     | Доступні функції |
|-----------|------------|------------|------------------|
| **Менеджер** | `manager`  | `manager`  | Повний доступ, перегляд прибутку, редагування меню |
| **Офіціант** | `waiter1`  | `waiter1`  | Робота з активними замовленнями на своїх змінах |
| **Клієнт** | `альбіна`   | `альбіна`   | Створення замовлення, перегляд історії своїх чеків |

## Технічні деталі
- **Framework:** ASP.NET Core 6.0/7.0 (MVC)
- **Database:** MS SQL Server / LocalDB
- **ORM:** Entity Framework Core (підхід Code First)
- **Розгортання:** База даних та початкові дані (Seeding) створюються автоматично при першому запуску додатку через `DBInitializer`.
